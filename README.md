## Currently building

**[EJC Hub](https://github.com/pahferreira/ejc-hub)** — A full-stack event management platform I use as an architectural playground. Every structural decision is intentional.

Key design decisions:
- **4-layer API module structure** — HTTP → Application → Core → Domain, with Single Responsibility enforced at every boundary
- **Dependency Inversion without a DI container** — core logic depends on repository interfaces, injected through a lightweight application factory
- **Shared permissions library** — single source of truth for authorization logic, consumed by both API and frontend
- **Type inference over duplication** — models from Drizzle schemas, forms from Zod schemas, permissions from const objects

This is where I test ideas before I bring them to production systems.
